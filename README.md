# Java-OOP-course

## Sections:
- Fundamentals:
  * String, Math, Random, Arrays classes
  * Abstract classes, Interfaces, Enums
  * Class hierarchy
  * Access modifiers
- Collections:
  * Java Collections and Maps
  * Generics
  * Exceptions
- Files:
  * Streams
